3a.4 TO EVALUATE INTEGRALS OF THE FORM

\(\int \frac{a \sin x + b \cos x}{c \sin x + d \cos x}\, dx\)
(di mana a, b, c, d adalah konstanta)

Berdasarkan buku halaman 59–66 • Introduction to Integral Calculus (Rohde et al., 2012)

A. Pemahaman Konsep

Di bagian 3a.4 ini, kita belajar menangani integral rasional trigonometri yang bentuk pembilang (numerator) dan penyebut (denominator) sama-sama kombinasi linear dari sin x dan cos x.

Intuisi: Mirip dengan integral bentuk \(\int \frac{a e^x + b}{c e^x + d}\, dx\) yang sudah kita pelajari di contoh sebelumnya. Kita memecah pembilang menjadi dua bagian:

  • Bagian yang sama persis dengan denominator
  • Bagian yang sama dengan turunan dari denominator

Ini membuat integral langsung terurai menjadi integral konstan + integral bentuk \(\int \frac{f'(x)}{f(x)}\, dx = \log |f(x)| + c\).

B. Dasar Teori (Buku hal. 59–60)

Metode ini adalah aplikasi langsung dari Corollary 4 dan Corollary 7 yang sudah dibahas di 3a.3.1.

Kita tulis pembilang sebagai:

\(a \sin x + b \cos x = A (c \sin x + d \cos x) + B \frac{d}{dx}(c \sin x + d \cos x)\)

Di mana turunan denominator adalah:

\(\frac{d}{dx}(c \sin x + d \cos x) = c \cos x - d \sin x\)

Setelah mencari A dan B dengan menyamakan koefisien sin x dan cos x, integral menjadi:

\(A \int dx + B \int \frac{c \cos x - d \sin x}{c \sin x + d \cos x}\, dx = A x + B \log |c \sin x + d \cos x| + c\)

C. Strategi Penyelesaian

Metode ini dipilih ketika:

  • Pembilang dan penyebut sama-sama linear kombinasi sin x dan cos x
  • Tidak ada cara substitusi langsung yang sederhana
  • Kita ingin mengubahnya menjadi bentuk yang langsung menggunakan logaritma (Corollary 4)
Contoh 16 (Buku hal. 60) — Example (16)

Hitung \( I = \int \frac{\sin x + 2 \cos x}{3 \sin x + 4 \cos x}\, dx \)

Turunan denominator: \( 3 \cos x - 4 \sin x \)

Tulis: \(\sin x + 2 \cos x = A(3 \sin x + 4 \cos x) + B(3 \cos x - 4 \sin x)\)

Sama kan koefisien:

\( (3A - 4B) \sin x + (4A + 3B) \cos x = 1 \sin x + 2 \cos x \)

Sistem persamaan:
\( 3A - 4B = 1 \)  (1)
\( 4A + 3B = 2 \)  (2)

Selesaikan: Kalikan (1) dengan 3 dan (2) dengan 4, lalu tambah → \( 25A = 11 \) → \( A = \frac{11}{25} \)
Masukkan ke (2) → \( B = \frac{2}{25} \)

Integral menjadi:

\( I = \frac{11}{25} \int dx + \frac{2}{25} \int \frac{3 \cos x - 4 \sin x}{3 \sin x + 4 \cos x}\, dx \)

\( = \frac{11}{25} x + \frac{2}{25} \log |3 \sin x + 4 \cos x| + c \)

Contoh 17 (Buku hal. 60) — Example (17)

Hitung \( I = \int \frac{1}{1 + \tan x}\, dx \)

Rewrite: \( \frac{1}{1 + \tan x} = \frac{\cos x}{\cos x + \sin x} \)

Turunan denominator: \( -\sin x + \cos x \)

Tulis: \( \cos x = A(\cos x + \sin x) + B(\cos x - \sin x) \)

Sama kan: \( (A + B)\cos x + (A - B)\sin x = 1 \cdot \cos x + 0 \cdot \sin x \)

→ \( A + B = 1 \)
\( A - B = 0 \)
→ \( A = \frac{1}{2} \), \( B = \frac{1}{2} \)

Integral menjadi:

\( I = \frac{1}{2} \int dx + \frac{1}{2} \int \frac{\cos x - \sin x}{\cos x + \sin x}\, dx \)

Substitusi \( \cos x + \sin x = t \Rightarrow (\cos x - \sin x)\, dx = dt \)

\( I = \frac{x}{2} + \frac{1}{2} \log | \cos x + \sin x | + c \)

F. Latihan Tambahan (Buku hal. 61)

(i) \(\int \frac{1}{1 + \cot x}\, dx = \frac{x}{2} - \frac{1}{2} \log |\sin x + \cos x| + c\)

(ii) \(\int \frac{1}{3 + 2 \tan x}\, dx = \frac{3}{13} x + \frac{2}{13} \log |3 \cos x + 2 \sin x| + c\)

(iii) \(\int \frac{3 \sin x + 4 \cos x}{2 \sin x + \cos x}\, dx = 2x + \log |2 \sin x + \cos x| + c\)

G. Refleksi & Insight Penting

  • Kenapa metode ini kuat? Karena langsung memanfaatkan turunan denominator sehingga muncul bentuk \(\frac{f'}{f}\).
  • Kesalahan umum: Lupa menuliskan turunan denominator dengan tanda minus yang benar (c cos x – d sin x).
  • Hubungan dengan bab sebelumnya: Ini adalah perluasan dari Generalized Power Rule dan Corollary 4 & 7 di 3a.3.
  • Aplikasi lanjut: Metode yang sama juga bisa dipakai untuk bentuk yang lebih umum (a + b cos x + c sin x) di bab berikutnya.

Tujuan akhir: Anda sekarang mampu menyelesaikan semua integral rasional trigonometri linear hanya dengan mencari A dan B.
